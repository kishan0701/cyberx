# 🛡️ CyberX — Cybersecurity Awareness Dashboard

A modern, interactive cybersecurity awareness platform designed to educate 
users about online threats, safe practices, and digital security.

## ✨ Features

- 🔴 **Live Threat Feed** — Real-time security incident updates from The Hacker News
- 🧠 **Security Quiz** — Test your cybersecurity knowledge interactively
- 🎣 **Phishing Simulator** — Learn to identify phishing attempts
- 🔐 **Password Strength Checker** — Analyze and strengthen your passwords
- 📊 **Research Dashboard** — Cybersecurity awareness survey & statistics
- 🛒 **Security Tools & Resources** — Curated tools and educational videos
- 📞 **Support Center** — Emergency contacts and cybersecurity helplines
- 📬 **Admin Inbox** — Secure admin panel for managing user messages

## 🛠️ Tech Stack

| Layer | Tech |
|-------|------|
| Frontend | HTML5, CSS3, Vanilla JS |
| Backend | Node.js, Express.js |
| Database | SQLite |
| Icons | Font Awesome |
| Charts | Chart.js |
